INTRODUCTION
---------------------

A plugin for showing wolrd clock.

REQUIREMENTS
---------------------

* No external module\library is required.

INSTALLATION
---------------------

Install as you would normally install a Druapl module.
Visit: https://www.drupal.org/docs/8/extend/installing-modules for further informtaion.


CONFIGURATION
---------------------

* Go to Administration > Configuration > Specbee World Clock.
  Set country, city and timezone for the clock.
